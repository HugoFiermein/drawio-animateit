# AnimateIT! — плагин для draw.io

Лёгкий плагин для draw.io, который анимирует соединители (линии) при выделении. При клике на линию она подсвечивается с эффектом бегущего потока, поднимается над перекрывающими её линиями, и возвращается к исходному виду сразу после снятия выделения.

Создан для сетевых диаграмм, схем стоек и любых диаграмм, где линии пересекаются и их сложно визуально отследить.

---

## Возможности

- **Анимация при выделении** — клик на любую линию запускает эффект бегущего потока вдоль неё
- **Подъём на передний план** — выделенная линия поднимается над всеми пересекающимися, и её всегда видно
- **Визуальное выделение** — линия становится толще (`strokeWidth=4`) и ярко-зелёной (`#00FF44`)
- **Без изменений в диаграмме** — исходный стиль полностью восстанавливается при снятии выделения
- **Совместимость с перемещением** — анимация корректно снимается при перетаскивании линии
- **Включение/выключение** — плагин можно отключить в любой момент через меню или горячую клавишу, без перезапуска draw.io

---

## Горячая клавиша

| Действие | Сочетание |
|---|---|
| Включить / выключить AnimateIT! | `Ctrl + Alt + I` |

Также можно переключить через **Дополнительно → AnimateIT!** в строке меню (✅ — включён, ⬜ — выключен).

## Установка — Десктопное приложение

Начиная с версии 19.0.3 внешние плагины в draw.io Desktop отключены по умолчанию. Для их использования нужно запустить приложение со специальным флагом.

### Шаг 1 — Скачайте плагин

Скачайте `AnimateIT.js` из этого репозитория и сохраните в удобное постоянное место, например:

```
C:\Users\ИмяПользователя\drawio-plugins\AnimateIT.js
```

### Шаг 2 — Включите внешние плагины

**Вариант А — Через ярлык:**

1. Правый клик на ярлыке draw.io → **Свойства**
2. В поле **Объект** добавьте флаг в конец строки:

```
"C:\Program Files\draw.io\draw.io.exe" --enable-plugins
```

3. Нажмите **OK** и всегда запускайте draw.io через этот ярлык

**Вариант Б — Через командную строку:**

```cmd
"C:\Program Files\draw.io\draw.io.exe" --enable-plugins
```

### Шаг 3 — Подключите плагин

1. Откройте draw.io (через изменённый ярлык или команду выше)
2. Перейдите в **Дополнительно → Плагины...**
3. Нажмите **Добавить**
4. Выберите локальный файл `AnimateIT.js`
5. Нажмите **OK**, затем **Применить**
6. Перезапустите draw.io — плагин будет загружаться автоматически при каждом запуске

> **Важно:** draw.io всегда нужно запускать с флагом `--enable-plugins` — без него плагин не загрузится.

## Установка — Веб-версия (app.diagrams.net)

Файл плагина должен быть доступен по публичному URL (например, через GitHub Pages).

1. Скопируйте `AnimateIT.js` в свой GitHub-репозиторий
2. Включите **GitHub Pages** (Settings → Pages → Branch: main)
3. URL плагина будет вида:
   ```
   https://<ваш-логин>.github.io/<репозиторий>/AnimateIT.js
   ```
4. Откройте [app.diagrams.net](https://app.diagrams.net)
5. Перейдите в **Дополнительно → Плагины → Добавить**
6. Вставьте URL и нажмите **OK → Применить**
7. Перезагрузите страницу браузера — плагин активен
